# 🎨 Pixel Art Creator

An interactive web application for creating pixel art using a customizable drawing grid.

## 🚀 Live Demo
https://mannatmarwaha.github.io/pixel-art-creator/

## 📂 GitHub Repository
https://github.com/mannatmarwaha/pixel-art-creator

## ✨ Features
- 🖌️ Pencil tool
- 🧹 Eraser tool
- 🎨 Multiple color palette
- 🌈 Custom color picker
- 📏 Grid size selection (16×16, 32×32, 64×64)
- 📥 Download artwork as PNG

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript

## 📖 About the Project
This project was created to practice JavaScript DOM manipulation, HTML Canvas, and responsive web design. It allows users to create pixel art and export their drawings as PNG images.
